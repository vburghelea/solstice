# ADR-2025-12-26-d0-10-member-directory-exposure: Member directory exposure

Status: Accepted
Date: 2025-12-26

## Decision

  and rationale.

## Options considered

- See the D0.10 section in `docs/sin-rfp/5.2-pro-review-output/d0-decision-analysis.md`.

## Rationale

- Aligns with the recommended choice documented in the D0 analysis.

## Consequences

- Requires implementation updates and tests per the consolidated backlog.
