# ADR-2025-12-26-d0-2-reporting-null-org-visibility: Reporting null-org visibility

Status: Accepted
Date: 2025-12-26

## Decision

Treat reporting cycles/tasks without organizationId as admin-only by default, with a roadmap for explicit assignment.

## Context

- This decision maps to D0.2 in `docs/sin-rfp/5.2-pro-review-output/consolidated-backlog.md`.
- See `docs/sin-rfp/5.2-pro-review-output/d0-decision-analysis.md` for options
  and rationale.

## Options considered

- See the D0.2 section in `docs/sin-rfp/5.2-pro-review-output/d0-decision-analysis.md`.

## Rationale

- Aligns with the recommended choice documented in the D0 analysis.

## Consequences

- Unblocks Stream C work.
- Requires implementation updates and tests per the consolidated backlog.
