# Tone and Style Notes

- Formal, confident, evaluator-friendly.
- Lead with the claim, then explain how it is met.
- Use short paragraphs and concrete statements.
- Avoid marketing fluff or vague claims.
- Cite evidence with file paths when available.

# Response Pattern

- State the position or approach in the first sentence.
- Follow with 2-4 sentences of specifics and scope.
- End with evidence references or assumptions.
