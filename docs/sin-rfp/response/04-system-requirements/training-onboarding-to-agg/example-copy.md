# Tone and Style Notes

- Formal, precise, evaluator-friendly.
- Use the matrix format as the primary response.
- Keep each row short and scannable.
- Mark Partial clearly and note the gap.
- Cite evidence with file paths in each row.

# Row Pattern

- Response: Comply / Configurable / Custom / Partial
- How requirement is met: 1-2 sentences.
- Evidence: file path or appendix reference.
- Notes: assumptions or dependencies.
